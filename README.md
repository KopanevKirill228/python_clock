## 🧭 О проекте

**Clock App** — это десктопное приложение с удобным графическим интерфейсом,
которое объединяет секундомер, будильник с базой данных, часы мирового времени
и ряд дополнительных утилит. Весь пользовательский интерфейс построен на PyQt5,
данные будильников хранятся в SQLite.

---

## ⚙️ Функциональность

### ⏱ Секундомер
- Запуск по нажатию кнопки **Старт**
- Отображает количество секунд, прошедших до нажатия **Стоп**

### 🔔 Будильник
- Все будильники сохраняются в БД (таблица `alarmclock`) и срабатывают в назначенное время
- По умолчанию воспроизводится стандартная мелодия
- Поддержка **кастомной музыки** — выбор файла через диалоговое окно (`QFileDialog`)
- Кнопка **остановки музыки** во время воспроизведения
- Кнопка **отмены** — удаляет последний будильник из БД (защита от случайного запуска)

### 🌍 Мировое время
- Одновременный показ текущего времени в **4 городах**
- Обновление каждую секунду через `QTimer` + библиотека `datetime`

### ✨ Особые функции
- 🖼 Картинки будильника и автора
- 📅 Отдельная форма с **календарём**
- 👤 Форма **«Об авторе»** с информацией о разработчике

## 🖥 Использование

| Функция              | Действие                                              |
|----------------------|-------------------------------------------------------|
| Секундомер           | **Старт** → засекает время → **Стоп** → показывает результат |
| Создать будильник    | Укажите время → нажмите **Добавить**                  |
| Сменить мелодию      | Кнопка **Выбрать музыку** → диалог выбора файла       |
| Остановить музыку    | Кнопка **Стоп** (активна во время воспроизведения)    |
| Отменить будильник   | Кнопка **Отмена** — удаляет последнюю запись из БД    |
| Мировое время        | Открывается автоматически, обновляется каждую секунду |
| Календарь            | Кнопка **Календарь** — открывает отдельную форму      |
