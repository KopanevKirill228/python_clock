⏰ Clock App
Многофункциональное приложение для работы со временем, реализованное на Python с использованием PyQt.

🧭 О проекте
Clock App — это десктопное приложение с графическим интерфейсом, объединяющее в себе секундомер, будильник с базой данных, мировые часы и дополнительные утилиты. Проект создан с акцентом на удобство использования и расширяемость.

⚙️ Функциональность
1. ⏱ Секундомер
Запуск и остановка по нажатию кнопки

Отображает точное время (в секундах), прошедшее между двумя нажатиями

2. 🔔 Будильник
Все будильники сохраняются в базу данных (таблица alarmclock)

Срабатывают точно в назначенное время

По умолчанию воспроизводится стандартная мелодия

Поддержка кастомной музыки — выбор файла через диалоговое окно

Кнопка остановки музыки во время воспроизведения

Кнопка отмены последнего будильника (на случай случайного запуска — удаляет последнюю запись из БД)

3. 🌍 Мировое время
Отображение текущего времени в 4 городах одновременно

Обновление каждую секунду через QTimer

Используется библиотека datetime для получения реального времени

4. ✨ Особые функции
🖼 Картинки будильника и информация об авторе

📅 Отдельная форма с календарём

👤 Форма «Об авторе» с информацией о разработчике

🛠 Технологии
Технология	Назначение
Python 3	Основной язык разработки
PyQt5	Графический интерфейс (GUI)
SQLite	Хранение будильников (alarmclock)
datetime	Получение текущего времени
QTimer	Периодическое обновление времени
QFileDialog	Выбор музыкального файла
